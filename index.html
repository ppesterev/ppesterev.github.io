CSS tabs

Basically, goes like this:

    <input id="tab-{number}">
    <label for"tab-{number}">
    [repeat a few times]
    
    <div class="tab-content"> (as many as there are inputs)

This is a bit scuffed because you can't just add more tabs without touching CSS (or putting in a silly amount of selectors in advance), but it seems to be the easiest way if all you want is some nice quick tabs with no JS.

Here's how it goes:

    /* Hide the radio buttons */
    
    input.tab-select {
        display: none;
    }
    
    /* Style the tab label */
    
    input.tab-select + label {
        ...
    }
    
    /* Some visual indication of hover could be nice */
    
    input.tab-select + label:hover {
        ...
    }
    
    /* Selected tab */
    
    input.tab-select:checked + label,
    input.tab-select:checked + label:hover {
        ...
    }
    
    /* Style the content box */
    
    input.tab-select ~ .tab-content {
        display: none;
        ...
    }
    
    /* When the first input is checked, unhide the first content box, etc */
    
    input:nth-of-type(1):checked ~ .tab-content:nth-of-type(1),
    input:nth-of-type(2):checked ~ .tab-content:nth-of-type(2),
    ... {
        display: block;
    }
